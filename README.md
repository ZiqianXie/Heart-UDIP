Main structure of this repo
1.data preprocessing pipe line which includes the functions like dicom2nii, nii_cropping, registration_template and image_registrations with ANTs.
2.the training codes of our cine MAE
3.the evaluation codes like SSIM and PNSR calculation
4.the interpretabilty codes like our Perturbation-based image reconstruction
